Développement d'un Écosystème de Surveillance Réseau

Ce projet consistait à concevoir une suite logicielle interconnectée permettant la détection, le stockage et la visualisation des failles de sécurité sur un réseau local.

Le cœur du système repose sur une application Java développée pour scanner le réseau, identifier les vulnérabilités actives et automatiser leur enregistrement au sein d'une base de données centralisée. Cette étape a nécessité la mise en place d'une logique d'automatisation des outils de diagnostic réseau.

L'interface de consultation se décline sur deux supports. D'une part, un site web dynamique interroge la base de données pour présenter les rapports de sécurité. D'autre part, une application Android native, développée en Java, permet de visualiser ces mêmes alertes en temps réel.

L'aspect critique du projet portait sur la communication entre ces différentes briques. L'échange de données entre le serveur web et l'application mobile a été réalisé via des API au format JSON, garantissant une interopérabilité fluide et efficace entre les plateformes fixe et mobile.
