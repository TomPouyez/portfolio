Pendu Football

Un jeu de pendu sur les clubs de football réalisés avec Pygame.

Jeu de Pendu sur le Football

Ce projet est un jeu de pendu réalisé avec Pygame sur le thème des clubs de football. Le but du jeu est de deviner le nom du club en un nombre limité d’essais. Le joueur interagit avec une interface visuelle animée et reçoit un retour immédiat sur ses choix.

Ce projet m’a permis de consolider mes compétences en programmation Python, gestion des événements, et interface utilisateur avec Pygame.
